## 🎯 使用方式

### 启动程序
```bash
# Windows: 双击 run.bat
# 或命令行：
cd src
python main.py
```

### 基本流程
1. 选择Word文档
2. 勾选检查规则
3. 开始检查
4. 查看结果
5. 导出报告

### 导出格式
- **Excel (.xlsx)** - 适合归档和数据分析
- **HTML (.html)** - 适合在线查看和分享
- **JSON (.json)** - 适合系统集成

## 🔧 技术栈

- **Python 3.8+**
- **PyQt5** - 桌面UI框架
- **python-docx** - Word文档处理
- **openpyxl** - Excel文件生成

## 📊 检查能力

### 自动检查（约30项）
程序可自动识别和检查的问题：
- 页面设置（纸张大小、边距）
- 签署完整性
- 公司名称
- 标题层级
- 表格和图标题存在性
- 计量单位格式
- 数学符号规范
- 敏感词

### 半自动检查（约15项）
需要结合程序检查和人工判断：
- 字体和字号
- 日期格式
- 数字格式
- 引用文件格式
- 公式完整性

## 📝 配置说明

所有检查规则都在 `config/rules.json` 中配置：

```json
{
  "id": "规则ID",
  "name": "规则名称",
  "description": "规则描述",
  "category": "所属类别",
  "enabled": true,
  "checker": "检查器类名",
  "params": {
    // 规则参数
  }
}
```

## 🚀 扩展开发

### 添加新检查器

1. 在 `src/core/checkers/` 创建检查器文件
2. 继承 `BaseChecker` 类
3. 实现 `check()` 方法
4. 在 `__init__.py` 中注册
5. 在 `rules.json` 中配置

### 自定义导出格式

参考 `src/utils/result_exporter.py` 实现新的导出方法。
